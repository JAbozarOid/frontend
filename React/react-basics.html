Navigation in React In a traditional website, when the button is clicked, the browser will send a POST request to the web server. The web server will return a new web page containing the button and movie name, and the web browser renders the new page. In a Single Page Application, when the button is clicked, the browser will send a POST request to a web server. The web server will return a JSON object. The application reads the object and updates the Label with the movie name.

key concepts in React Navigation
* Bundling: The process of loading all necessary HTML, CSS, and JavaScript resources at once when the application is first requested.
* Navigation: The method by which users move between different views or components in an application, which differs significantly between SPAs and traditional web apps.
* Lazy Loading: A technique where only the minimum resources needed to load the application are fetched initially, with additional resources loaded as required.
* Single Page Application (SPA): A web application that interacts with the user by dynamically rewriting the current page rather than loading entire new pages from the server.
