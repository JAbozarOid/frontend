In general, there are two ways to export modules in JavaScript:

  1. Using default exports : default exports, you can export as many items from any JavaScript file as you want
  2. Using named exports : Named exports are a way to export only certain parts of a given JavaScript file.

export default function addTwo(a, b) { console.log(a + b); }

function addTwo(a, b) { console.log(a + b); }

function addThree(a, b, c) { console.log(a + b + c); }

export { addTwo, addThree }; Using named exports

import module import addTwo from "./addTwo"; which was exported by export default

import module import { addTwo } from "./addTwo"; which was exported by named exports
