TypeScript : is a JavaScript with Type Checking

TypeScript benefits

  • static typing
    • let x = 10 -> x is an integer and can be reassigned to x ="a" -> now x is character -- type not defined
    • let x : number = 10 -> x is an integer or number type and can not to reassign to x ="a" (runtime error) - type defined
  • code completion
  • refactoring
  • shorthand notations

Browser support: html / css / javascript

  • typescript files (.ts) compiled to javascript (.js)

TypeScript usage

  • Medium to large project

JavaScript usage

  • Simple project

Install TypeScript (our .ts file should compile to .js JavaScript)

  • npm i -g typescript --> ( i stand for install)
  • tsc --v (get the version od typescript which is install in out system (typescript compiler version)
  • tsc main.ts (this command compile the current file)
  • for every files it is time consuming and hard to compile or transpile each .ts file to .js files so use this command : tsc --init (will create a tsconfig.json)
  • TypeScript annotations
    • number
    • string
    • boolean
    • any (let x : any = 10 --> x = "a" --> x = true --> reassign every things)
  • tupels
    • it like array but when defined should say what is the exact size of it
    • tupels is fixed size array
